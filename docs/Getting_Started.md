# Getting started — NM i AI 2026

Welcome to the **NM i AI 2026** documentation. The competition runs from **19 March at 18:00 CET** to **22 March at 15:00 CET** — 69 hours of challenges with a **1,000,000 NOK** prize pool.

## The three tasks

NM i AI 2026 has **three independent** AI challenges. Your **overall score** is the average of your normalized scores on all three tasks (**33% each**).

| Task | Type | Submission |
|------|------|------------|
| **Tripletex** | AI accounting agent | HTTPS API endpoint |
| **Astar Island** | Norse world prediction | REST API predictions |
| **NorgesGruppen Data** | Grocery shelf object detection | Code upload (ZIP) |

## Quick start

1. Sign in at **[app.ainm.no](https://app.ainm.no)** with Google.  
2. Create or join a team (max **4** members).  
3. Complete **Vipps** verification for prize eligibility.  
4. Explore task documentation in the sidebar.  
5. Submit to as many tasks as you can.

## MCP server

Connect this docs server to Claude Code (or compatible tools) for AI-assisted development:

```bash
claude mcp add --transport http nmiai https://mcp-docs.ainm.no/mcp
```

## Need help?
